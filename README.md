# travel-alya
Just another project

## Setting up

### Pulling repositories

```
git clone https://github.com/kofix-ml/travel-alya.git
```

### setting up project

1. go into project folder

```
cd travel-alya/
```

2. initialising project

```
composer install
npm install
```

3. setting up env

```
cp env .env
```
